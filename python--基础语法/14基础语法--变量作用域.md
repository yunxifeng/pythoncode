# 变量作用域
- 变量有作用范围限制
- 分类：变量按作用域分类
     - 全局（global）：在函数外部定义
     - 局部（local）：在函数内部定义
- 变量的作用范围
     - 全局变量：在整个全局范围都有效
         - 全局变量在局部可以使用（即函数内部可以访问函数外部定义的变量）
     - 局部变量在局部范围可以使用
         - 局部变量在全局范围无法使用
- LEGB原则
     - L（local）局部作用域
     - E（Enclosing function locale）外部嵌套函数作用域
     - G（Global module）函数定义所在模块作用域
     - B（Buildin）：Python内置模块的作用域
