# 元组(Tuple)
- 可以看作是一个不可更改的list
## Tuple创建
- 注: 当创建单元素元组时,不能省略/忘记"," 
## Tuple特性
   - 有序列表，有序
   - 元组数据值可以访问，不能修改
   - 元组数据可以是任意类型
   - 总之，list所有特性，除了可修改外，元组都具有
   - 也就意味着，list具有的一些操作，比如索引，分片，序列相加，相乘，成员资格操作等，一模一样
## Tuple常见操作
   - 1.索引操作
   - 2.分片操作
   - 3.Tuple相加
       - 注: 传址操作
   - 4.Tuple乘法
       - t * n
   - 5.成员检测
   - 6.Tuple遍历
## 关于Tuple的函数
   - 1.len(t): 获取元组的长度
   - 2.最值
       - max(t): 最大值
       - min(t): 最小值
   - 3.转化或创建Tuple
       - t = tuple(*) 
   - 4.t.count(*): 查找指定元素的出现次数
   - 5.t.index(*): 查找指定元素在Tuple中的索引值
       - 注: 如果需要查找的数字是多个，则返回第一个
